# Piercing Featured Images

Bu klasordeki carousel gorselleri icin hedef format:

- Format: WebP
- Renk uzayi: sRGB
- Kalite: q=75-85
- Onerilen genislik: 1200px+
- Oran: serbest (kartta `contain` ile sigdirilir)

Not: Gorsel eksik veya yuklenemezse kart media alaninda placeholder gradient gorunur.
